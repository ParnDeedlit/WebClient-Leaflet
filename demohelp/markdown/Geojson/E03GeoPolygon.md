#### GeoJSON-Polygon
------

***注：*** 对类型"Polygon"来说，"coordinates"成员必须是一个线性
环坐标数组的数组。对拥有多个环的的面来说，第一个环必须是
外部环，其他的必须是内部环或者孔。

标准数据格式：
```javascript
var GeoPolygon = {
        "type": "Feature",
        "properties": {},
        "geometry": {
            "type": "Polygon",
            "coordinates": [
                [
                    [
                        114.32510375976562,
                        30.63968439181164
                    ],
                    [
                        114.22931671142578,
                        30.608958829007946
                    ],
                    [
                        114.22348022460937,
                        30.601275914486066
                    ],
                    [
                        114.22931671142578,
                        30.580588116155223
                    ],
                    [
                        114.32510375976562,
                        30.63968439181164
                    ]
                ]
            ]
        }
    };
```
