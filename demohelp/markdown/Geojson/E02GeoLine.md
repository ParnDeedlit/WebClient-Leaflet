### GeoJSON-Line

```text
对类型"LineString"来说，“coordinates"成员必须是两个
或者多个位置的数组。
```

> * 标准数据格式：

```javascript
var GeoLine = {
        "type": "LineString",
        "properties":{
            "prop0": "value0"
        },
        "coordinates": [
                [
                    114.34776306152344,
                    30.623436511269382
                ],
                [
                    114.32510375976562,
                    30.63909360759635
                ],
                [
                    114.30673599243164,
                    30.634958017061198
                ],
                [
                    114.29180145263672,
                    30.629640569460406
                ]]
    };
```

**注：**
线性环具有4个或者更多位置的封闭的线。第一个和最后一个位置是相等的（它们表示相同的的点）。
