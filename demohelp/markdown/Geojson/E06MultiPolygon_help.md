#### GeoJSON-MultiPolygon
------

> 对类型"MultiPlygon"来说，"coordinates"成员必须是`面坐标数组`的数组(即`四维数组`)。

##### 提交BUG
> 找到bug请提交，我们会及时解决[issue](https://github.com/ParnDeedlit/WebClient-Leaflet/issues)。

标准数据格式：
```javascript
var MultiPolygon = {
        "type": "Feature",
        "properties": {},
        "geometry": {
            "type": "MultiPolygon",
            "coordinates": [
                [
                    [
                        [
                            114.26811218261719,
                            30.622254740808902
                        ],
                        [
                            114.23618316650389,
                            30.552800413453546
                        ],
                        [
                            114.32853698730467,
                            30.519681272749402
                        ],
                        [
                            114.37076568603515,
                            30.587977122488372
                        ],
                        [
                            114.3563461303711,
                            30.616936595191674
                        ],
                        [
                            114.3240737915039,
                            30.63968439181164
                        ],
                        [
                            114.26811218261719,
                            30.622254740808902
                        ]
                    ]
                ]
            ]
        }
    };
```
> 请注意，在Leaflet中，绘制多面区域`不区分`顺逆方向。若在同一区域内绘制多个区，不管顺逆均会出现洞区。
